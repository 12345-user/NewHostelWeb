# 客栈网站内容资料清单

下面是后续补充网站内容时建议使用的标准格式。你可以按表格填写，也可以直接把同样字段发给我，我来整理进系统。

## 1. 客栈基础信息

| 字段 | 标准格式 | 示例 |
| --- | --- | --- |
| 客栈名称 | 文本 | 猫驼旅者客栈 |
| 一句话定位 | 20 字以内 | 大理古城里的旅人小院 |
| 简介 | 100-300 字 | 介绍客栈风格、位置、适合人群 |
| 地址 | 省 / 市 / 区 / 详细地址 | 云南省大理市古城内 |
| 联系电话 | 手机号 | 13800000000 |
| 微信 | 微信号或二维码图片 | hostel_owner |
| 邮箱 | 邮箱地址 | hello@example.com |
| 预订方式 | 链接 / 电话 / 微信 | 小程序链接或平台链接 |
| 入住时间 | HH:mm | 14:00 后 |
| 退房时间 | HH:mm | 12:00 前 |
| 重要提示 | 分条文本 | 禁烟、宠物、停车、发票等 |
| 首页主图 | 3 张，建议 1920x1080 | hero-1.jpg / hero-2.jpg / hero-3.jpg |

## 2. 活动标准格式

| 字段 | 标准格式 | 示例 |
| --- | --- | --- |
| 活动标题 | 文本 | 周末篝火故事会 |
| 活动日期 | YYYY-MM-DD | 2026-03-15 |
| 参与人员 | 逗号分隔 | 小雨, 阿杰, Lucy |
| 活动摘要 | 30-80 字 | 星空下的温暖相聚，分享旅途中的奇遇与感动 |
| 活动详情 | 100-800 字 | 活动过程、亮点、住客反馈、适合人群 |
| 活动图片 | 1-6 张，建议 16:9 | activity-1.jpg |
| 是否推荐首页 | 是 / 否 | 是 |

推荐提交模板：

```text
活动标题：
活动日期：
参与人员：
活动摘要：
活动详情：
活动图片：
是否推荐首页：
```

## 5. 故事 / 动态扩展格式

当前代码里“故事”可以先放在活动详情中，也可以后续单独扩展成博客栏目。

| 字段 | 标准格式 | 示例 |
| --- | --- | --- |
| 故事标题 | 文本 | 一个雨夜来的骑行客 |
| 发布时间 | YYYY-MM-DD | 2026-04-21 |
| 作者 | 文本 | 主理人 |
| 摘要 | 30-80 字 | 讲述一位住客在雨夜抵达的故事 |
| 正文 | 300-1500 字 | 完整故事内容 |
| 配图 | 1-6 张 | story-1.jpg |
| 关联活动 / 人员 | 可选 | 周末篝火故事会 / 小雨 |

## 6. 后台账号权限

| 角色 | 权限 | 账号数量 |
| --- | --- | --- |
| 负责人 owner | 新增、编辑、删除全部内容 | 1 |
| 管理员 admin | 新增、编辑内容，不可删除 | 3 |
| 未登录用户 | 只能访问公开展示页面 | 不限 |
